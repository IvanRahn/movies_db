# Movie database app

A Ruby on Rails app made as part of Coder Academy Bootcamp.
Allows CRUD of movies, populates some of the fields by connecting to the TMDB.

Ruby version 2.5.1
Rails version 5.2.1
Database used - PostgreSQL
